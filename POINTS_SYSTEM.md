# 💰 积分系统更新说明

## 📊 新规则概览

### 1. 挂机收益上限提升
- **旧规则**：离线最多获得 100 积分
- **新规则**：离线最多获得 **1000 积分**
- **收益速率**：每10秒获得1积分（不变）

### 2. 负分游戏机制
- **旧规则**：积分不足无法下注
- **新规则**：允许负分继续游戏
- **负分限制**：单次押注不超过 **1000 积分**

---

## 🎮 详细规则

### 挂机收益计算

#### 收益公式
```
离线时间（秒）÷ 10 = 获得积分（最多1000）
```

#### 示例
| 离线时间 | 获得积分 |
|---------|---------|
| 1分钟 | 6积分 |
| 10分钟 | 60积分 |
| 1小时 | 360积分 |
| 3小时 | 1000积分（已达上限） |
| 10小时 | 1000积分（已达上限） |

**结论**：离线3小时以上，都是1000积分

## 💸 负分游戏系统

### 如何进入负分？
1. 连续下注失败
2. 积分耗尽后继续下注
3. 例如：有50积分，押注100，失败后变成-50

### 负分状态下的规则

#### 押注限制
```
正分状态：可以押注全部积分
负分状态：单次押注 ≤ 1000
```

#### 示例场景

**场景1：正分押注**
```
当前积分：500
可押注范围：1-500
押注：500 ✅
```

**场景2：负分押注**
```
当前积分：-300
可押注范围：1-1000
押注：500 ✅
押注：1500 ❌（超过1000限制）
```

**场景3：微正分押注**
```
当前积分：50
可押注范围：1-50
押注：50 ✅
押注：100 ❌（超过当前积分）
```

### 翻身策略

#### 小额积累
```
负分：-500
押注：100，赔率3.5x
获胜：+350
结果：-150（继续努力）
```

#### 大额翻盘
```
负分：-500
押注：1000，赔率4.0x
获胜：+4000
结果：+3500（翻身成功！）
```

#### 持续失败
```
负分：-500
押注：1000，失败
结果：-1500（越陷越深）
```

## 🎯 策略建议

### 正分时（积分 > 0）

#### 保守策略
- 每次押注10-20%积分
- 选择胜率高的蛇（赔率低但稳定）
- 目标：稳定增长

#### 激进策略
- 每次押注50-100%积分
- 选择胜率低的蛇（赔率高但风险大）
- 目标：快速翻倍

### 负分时（积分 < 0）

#### 谨慎翻身
- 每次押注100-300
- 选择中等赔率（3-4倍）
- 慢慢回正

#### 孤注一掷
- 直接押注1000
- 选择高赔率蛇（4倍以上）
- 一把翻身或更深

### 零分附近（-100 ~ +100）

#### 最危险区域
- 正分不够押大注
- 负分压力增加
- 建议：小额多次，稳定回正

## 💡 风险管理

### 正分玩家
✅ 优势：
- 可以根据积分灵活押注
- 没有上限限制
- 容错率高

⚠️ 建议：
- 不要一次押注全部
- 保留至少20%作为后备
- 长期稳定增长

### 负分玩家
✅ 优势：
- 还能继续玩
- 有翻身机会
- 最多押1000限制风险

⚠️ 风险：
- 负分越多压力越大
- 容易心态失衡
- 可能越陷越深

📌 **建议**：
- 负分时选择中高赔率（3-4倍）
- 不要盲目押1000
- 分批翻身更安全
