# 🎲 随机天赋系统

## 核心机制

每局游戏开始时，系统会**随机分配**4种不同的天赋给4条AI蛇，确保：
- ✅ 每条蛇获得不同的天赋
- ✅ 每局游戏的技能组合都不同
- ✅ 增加游戏的不确定性和可玩性

## 🎯 四种天赋

### ⚡ 疾风步（速度型）
- 移动速度+15%
- 代价：每次吃食物只得5分（其他蛇10分）
- 策略：快速抢夺食物，灵活躲避

### 🛡️ 铁壁（防御型）
- 每吃5个食物获得1个护盾
- 护盾可抵挡一次致命碰撞
- 有护盾时头部显示金色光环
- 策略：稳扎稳打，容错率高

### 🔥 狂暴（成长型）
- 每吃1个食物速度+5%
- 最多叠加+50%速度
- 策略：滚雪球，越战越强

### 🧲 磁力场（控制型）
- AI倾向于占据地图中心位置
- 更容易获取中心区域的食物
- 策略：控制核心区域，稳定发展

## 🎮 玩家体验

### 下注前的观察
玩家可以通过以下方式了解本局天赋分配：
1. **分数栏图标**：每条蛇旁边显示天赋图标
2. **下注按钮**：按钮上显示蛇名+天赋图标
3. **悬停提示**：鼠标悬停在图标上查看详细说明

### 策略思考
- 不同的天赋组合会影响游戏走向
- 例如：疾风步蛇在前期有优势，狂暴蛇在后期更强
- 铁壁蛇容错率高，磁力场蛇位置优势明显

### 赔率变化
- 随着游戏进行，某些天赋的胜率会提高
- 赔率系统会根据历史胜率自动调整
- 玩家需要观察哪些天赋更强，做出明智的下注选择

## 🔧 技术实现

### 天赋池
```javascript
const TALENT_POOL = [
    { name: '疾风步', icon: '⚡', type: 'speed', ... },
    { name: '铁壁', icon: '🛡️', type: 'shield', ... },
    { name: '狂暴', icon: '🔥', type: 'rage', ... },
    { name: '磁力场', icon: '🧲', type: 'magnet', ... }
];
```

### 随机分配算法
使用 Fisher-Yates 洗牌算法：
```javascript
assignRandomTalents() {
    const availableTalents = [...TALENT_POOL];
    
    // 洗牌
    for (let i = availableTalents.length - 1; i > 0; i--) {
        const j = Math.floor(Math.random() * (i + 1));
        [availableTalents[i], availableTalents[j]] = 
            [availableTalents[j], availableTalents[i]];
    }
    
    // 分配给每条蛇
    SNAKE_CONFIG.forEach((config, index) => {
        this.currentTalents[config.name] = availableTalents[index];
    });
}
```

### 显示更新
```javascript
updateTalentDisplay() {
    SNAKE_CONFIG.forEach(config => {
        const talent = this.currentTalents[config.name];
        talentElement.textContent = talent.icon;
        talentElement.title = `${talent.name}：${talent.description}`;
    });
}
```

## 🎲 游戏变化

### 每局都不同
- 第1局：沫桐=⚡, 童谣=🛡️, 斌飞=🔥, 曹=🧲
- 第2局：沫桐=🔥, 童谣=🧲, 斌飞=🛡️, 曹=⚡
- 第3局：沫桐=🛡️, 童谣=⚡, 斌飞=🧲, 曹=🔥
- ... 24种可能的排列组合

### 策略深度
玩家需要：
1. 观察本局的天赋分配
2. 分析哪种天赋组合更强
3. 考虑当前的赔率
4. 做出最优的下注决策

## ✨ 优势

### 提升可玩性
- ❌ 旧系统：每条蛇天赋固定，玩家很快掌握规律
- ✅ 新系统：每局都不同，始终保持新鲜感

### 增加变数
- 防止某条蛇胜率过高
- 让每条蛇都有机会获胜
- 赔率更加平衡

### 观察乐趣
- 玩家需要仔细观察天赋图标
- 增加了策略思考的深度
- 每局游戏都有新的发现
