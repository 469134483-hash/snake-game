# 💳 欠债下注系统

## 核心机制

**积分不足时可以欠债下注，直接进入负分状态！**

不用担心积分用完无法游戏，你可以继续下注，只是会产生负债。

---

## 🎮 下注规则详解

### 正分状态（积分 ≥ 0）

#### 积分充足
```
当前积分：500
下注金额：300
结果：直接扣除，剩余200 ✅
```

#### 积分不足
```
当前积分：100
下注金额：500
系统提示：将产生400的负债，确定继续吗？
确认后：扣除500，剩余-400 ✅
```

### 负分状态（积分 < 0）

#### 限制押注上限
```
当前积分：-300
下注金额：500
结果：允许，扣除后-800 ✅

当前积分：-300
下注金额：1500
结果：拒绝！负分时单次押注≤1000 ❌
```

## 💡 策略建议

### 正分玩家（积分 > 0）

#### 何时欠债？
✅ **建议欠债**：
- 看到高赔率机会（4倍以上）
- 观察到某条蛇有明显优势
- 想要快速翻倍

❌ **不建议欠债**：
- 盲目追高
- 情绪化下注
- 连续失败后

#### 欠债额度建议
```
积分100，欠债建议：
- 保守：欠100-200（总押注200-300）
- 适中：欠300-500（总押注400-600）
- 激进：欠500-1000（总押注600-1100）
```

### 负分玩家（积分 < 0）

#### 翻身策略

**稳健型**：
```
负分：-500
押注：200-300
赔率：3-3.5x
目标：慢慢回正
```

**平衡型**：
```
负分：-500
押注：500-700
赔率：3.5-4x
目标：中等收益
```

**孤注一掷**：
```
负分：-500
押注：1000（上限）
赔率：4-5x
目标：一把翻身
```

## ⚠️ 风险管理

### 欠债的风险

#### 心理压力
- 负分会影响心态
- 容易做出冲动决策
- 可能陷入"追债"陷阱

#### 财务风险
- 负债越多越难翻身
- 需要更高赔率才能回本
- 连续失败会雪上加霜

### 如何避免深度负债

#### 1. 设定止损线
```
规则：负债超过1000就停止欠债
当前：-500 → 可以继续
当前：-1200 → 停止，等挂机恢复
```

#### 2. 分批翻身
```
错误：-1000，押注1000，失败 → -2000
正确：-1000，押注300×3次，逐步回正
```

#### 3. 利用挂机
```
负分严重时：
1. 停止下注
2. 挂机3小时获得1000
3. 回到可控范围再继续
```

## 📈 数学期望

### 欠债下注的期望值

假设平均赔率3.5倍，胜率25%：
```
期望 = 押注额 × (胜率 × 赔率 - 1)
     = 100 × (0.25 × 3.5 - 1)
     = 100 × (0.875 - 1)
     = 100 × (-0.125)
     = -12.5
```

**结论**：长期来看会亏损

但是：
- 可以通过观察天赋提高胜率
- 选择被低估的蛇
- 避开被高估的蛇
- 有可能实现正期望
