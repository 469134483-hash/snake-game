# 🐍 贪吃蛇对战游戏

一个使用原生 JavaScript 和 HTML5 Canvas 开发的贪吃蛇对战游戏。与AI对手竞争，看谁能吃到更多食物！

## 🎮 游戏特性

- ✨ 流畅的游戏体验
- 🤖 **AI对手**：智能的电脑控制蛇与你竞争
- 🎯 双方得分系统和最高分记录
- 🏆 实时对战：抢先吃到食物获得分数
- ⚡ 随着得分增加，游戏速度逐渐提升
- 🎨 现代化的 UI 设计
- 📱 响应式布局，支持移动端
- 💾 使用 LocalStorage 保存最高分

## 🚀 如何运行

### 方法一：直接打开
1. 双击 `index.html` 文件，在浏览器中打开

### 方法二：使用本地服务器（推荐）
1. 安装 Python（如果还没有安装）
2. 在项目目录下运行：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # 或 Python 2
   python -m SimpleHTTPServer 8000
   ```
3. 在浏览器中访问 `http://localhost:8000`

### 方法三：使用 VS Code Live Server
1. 在 VS Code 中安装 "Live Server" 扩展
2. 右键点击 `index.html`，选择 "Open with Live Server"

## 🎯 游戏玩法

- **方向键（↑ ↓ ← →）**：控制你的蛇（绿色）的移动方向
- **空格键**：暂停/继续游戏
- **开始游戏按钮**：开始新游戏
- **暂停按钮**：暂停当前游戏
- **重置按钮**：重置游戏到初始状态

## 📝 游戏规则

1. 🟢 **绿色蛇**：由你控制
2. 🔵 **蓝色蛇**：由AI控制，会智能寻找食物
3. 🔴 **红色食物**：抢先吃到食物得10分，蛇会变长
4. 💥 **游戏结束条件**：
   - 撞到墙壁
   - 撞到自己的身体
   - 撞到对手的身体
5. 🏆 **获胜条件**：对手先死亡，你就赢了！
6. ⚡ 随着得分增加，蛇的移动速度会加快

## 🛠️ 技术栈

- **HTML5**：页面结构
- **CSS3**：样式和动画
- **JavaScript (ES6+)**：游戏逻辑
- **Canvas API**：游戏渲染

## 🔧 自定义配置

你可以在 `game.js` 文件的 `CONFIG` 对象中修改游戏参数：

```javascript
const CONFIG = {
    CANVAS_SIZE: 400,        // 画布大小
    GRID_SIZE: 20,           // 网格大小
    INITIAL_SPEED: 150,      // 初始速度（毫秒）
    SPEED_INCREMENT: 5,      // 速度增量
    MIN_SPEED: 50           // 最小速度（最快）
};
```
